# Epigenomic Data Analysis: ChIP-seq Peak Calling

A bioinformatics workflow for analyzing ChIP-seq data using **MACS3** in Google Colab. This project performs quality control, peak calling, and basic visualization of enriched genomic regions.

## Features

* ChIP-seq data preprocessing
* Quality control with Samtools
* Peak calling using MACS3
* Peak visualization and summary analysis

## Tools & Libraries

* Python
* MACS3
* Samtools
* Pandas
* Matplotlib

## Platform

* Google Colab
